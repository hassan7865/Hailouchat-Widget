Chat Widget Integration Example

This is an example page showing how the chat widget can be integrated into any HTML page.

Note: The chat widget will appear in the center of this page. You can pass a client_id as a URL parameter like: ?client_id=your-client-id

Features:

  • ✅ Modular component architecture
  • ✅ Configurable client ID via URL parameters
  • ✅ Responsive design
  • ✅ Real-time WebSocket communication
  • ✅ Device and location detection
  • ✅ Typing indicators
  • ✅ Message history
  • ✅ Connection status indicators

Integration:

To integrate this widget into your website, simply include the injector script with your client ID.


<!-- Simple integration with injector script -->
<script 
  src="path/to/injector.js" 
  data-client-id="your-client-id">
</script>

<!-- Or with URL parameters -->
<script src="path/to/injector.js"></script>
<!-- Then visit: yoursite.com?client_id=your-client-id -->
